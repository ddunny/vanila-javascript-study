# TODO list

- 리스트 종류 : to do, doing, done (라벨이라고 부름)

1. 입력
   1. (기본) 텍스트
   2. 각 리스트 내에서 입력이 가능한 상태를 할지(각 리스트 상단에 + 버튼) **OR** 입력 폼을 따로 둘지 (버튼을 눌러서 모달로 뜬다던지)
2. 수정
   1. 텍스트
   2. 라벨 변경 (=상태 변경)
3. 삭제
